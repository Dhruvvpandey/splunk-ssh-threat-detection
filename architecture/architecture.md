# Splunk SSH Threat Detection Architecture

## Overview

This project uses Splunk SIEM to collect, process, analyze, and visualize Linux SSH authentication logs.

## Data Flow

Linux SSH Logs
↓
Splunk Index
↓
SPL Queries
↓
Security Dashboard
↓
Threat Detection & Investigation

## Components

### Linux SSH Logs

Authentication logs generated by Linux systems containing successful and failed login attempts.

### Splunk Index

Stores and organizes security events for analysis.

### SPL Queries

Custom searches used to detect:

* Failed logins
* Brute-force attacks
* Suspicious source IPs
* Successful authentications

### Dashboard

Visualizes security events and trends using Splunk dashboards.

## Security Use Cases

* SSH Brute Force Detection
* Failed Login Monitoring
* Attacker IP Identification
* Account Activity Monitoring
